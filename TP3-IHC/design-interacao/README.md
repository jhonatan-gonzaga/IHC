# IHC
## RESUMO

O projeto de Design de Interação foi estruturado para traduzir as intenções do sistema em experiências concretas de uso, começando pela definição da Metamensagem, que estabelece a comunicação implícita entre o designer e o usuário sobre a solução proposta. Com base nessa visão, foi desenvolvido um Mapa de Objetivos visuais para personas distintas — como Clientes, Profissionais e Donos de Loja — englobando metas que variam desde o gerenciamento de contas até a contratação de serviços e venda de materiais.

A partir desses objetivos, foram modeladas as tarefas do sistema utilizando a análise hierárquica (HTA), garantindo que cada um dos fluxos principais, como "Encontrar Profissional" ou "Gerenciar Notificações", estivesse diretamente associado e consistente com os objetivos mapeados. Para representar a dinâmica da interação, utilizou-se a linguagem MoLIC, detalhando a conversação entre usuário e sistema em diagramas que cobrem desde o cadastro inicial até fluxos complexos de suporte técnico. O trabalho foi finalizado com a elaboração de protótipos de baixa fidelidade (sketches), que tangibilizam visualmente as tarefas e interações planejadas nos HTAs e MoLICs, seguindo uma organização rigorosa de arquivos e diretórios.
