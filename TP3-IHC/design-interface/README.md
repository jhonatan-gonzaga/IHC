# IHC
## RESUMO

O projeto de Design de Interface foi estruturado para realizar a aplicação de todas as pesquisas e levantamentos anteriores, como Briefings, HTAs, MoLICs, Sketches etc. Formentando num sistema interativo rudimentar, porém que pode ser a base de um projeto muito maior no futuro.

A partir de tudo que coletamos, foi feito um mural de inspiração para tratar acerca das ideias que teríamos para a interface final do projeto, logo após, a paleta e tipografia foram definidas para traduzir melhor a mensagem do aplicativo, assim como seu ícone. E após criar algumas telas interativas no Figma, se baseando nas 10 Heurísticas de Nielsen, a interação inicial do nosso sistema foi feita.
